# RSV-A G Intrahost Haplotype Analysis

This repository contains the analysis notebooks, input files, processed tables, and figure panels for the RSV-A G-gene intrahost haplotype analysis.

## Repository Structure

```text
config/                   Analysis path configuration
data/input/               LoFreq iSNV calls, haplotype FASTA files, and consensus FASTA files
data/metadata/            Sample metadata and depth-QC tables
data/reference/           Reference FASTA, GFF annotation, and G-region reference alignment
data/processed_data/      Step-level tables generated by the notebooks
notebooks/                Methods-ordered Jupyter notebooks
results/figures/          Figure panels exported as PNG, PDF, and SVG
```

The workflow uses the files included in this repository:

## Environment

Create and activate the conda environment from the repository root:

```bash
conda env create -f environment.yml
conda activate rsva-g-haplotypes
```

## Notebook Workflow

Run the notebooks in `notebooks/` in numeric order.

1. `01_data_sources_and_sample_inclusion.ipynb`
2. `02_read_preprocessing_and_reference_alignment.ipynb`
3. `03_consensus_sequence_generation_and_clade_annotation.ipynb`
4. `04_intrahost_variant_calling.ipynb`
5. `05_screening_for_covarying_g_gene_sites.ipynb`
6. `06_g_gene_haplotype_reconstruction.ipynb`
7. `07_g_gene_phylogeny_and_haplotype_branch_assignment.ipynb`
8. `08_permutation_test_for_cross_branch_combinations.ipynb`
9. `09_identification_of_branch_defining_sites.ipynb`
10. `10_de_novo_mutation_plausibility_calculation.ipynb`
11. `11_global_consensus_g_gene_background_analysis.ipynb`

Each notebook writes its tables to `data/processed_data/<step>/`. Notebooks that generate figures write PNG, PDF, and SVG files to `results/figures/<step>/`.
